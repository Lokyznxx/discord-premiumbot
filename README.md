<p align="center">Um Bot para o Discord desenvolvido com <a href="https://discord.js.org/#/">discord.js</a></p>

## Funcionalidades:

* Sistema de multi linguagens
* Sistema de Premium


## Comandos:

* !ping

* !setlang

* !keygen

* !premium

* !claim

* <a href="https://nodejs.org/en/">Node.js</a> versão 16.9 ou superior
